# 🎮 ゲーム企画書「目だらけハンバーグ 〜どうぞ召し上がれ〜」

## 📝 ストーリー原案（子供の音声入力から）

> このゲームは、ハンバーグを作るゲームです。ハンバーグには目がたくさんあります。でもそれは食べなきゃいけません。でもそれは世界一美味しいので大丈夫。そしてこのゲームは楽しくすごせば大丈夫。でもみんなで楽しく過ごすのが一番。そして、今話題のモスバーガー選べるよ。ポテトを作るかハンバーガーを作るかお肉を焼くかチーズを盛りつけ野菜とかをつけるか、それかモスバーガーに行ってハンバーガーを作るか。それは美味しくいただいてください。どうぞ召し上がれ。

## 🎯 Claude Codeへの指示

上記のストーリーをもとに、以下の仕様でブラウザゲームを作ってください。

### ゲーム基本設定

- **タイトル**: 「目だらけハンバーグ 〜どうぞ召し上がれ〜」
- **ゲーム形式**: ブラウザで遊べるHTML+CSS+JavaScript（ファイル分割構成）
- **ジャンル**: クッキングシミュレーション
- **対象年齢**: 子供向け（小学生）
- **操作方法**: クリック／タップ（スマホ・タブレット対応）
- **プレイ時間目安**: 10〜20分（何度でも繰り返し遊べる）
- **雰囲気**: 楽しくてちょっとシュール。ポップでカラフル

### 技術要件

- 外部ライブラリ不要（Canvas APIまたはDOM操作で実装）
- レスポンシブ対応（PC・タブレット両方で遊べる）
- 日本語テキスト
- 絵文字ベースのビジュアル（🍔🍳🥩🧀🥬🍟👁️🫣）

### ファイル構成（適度に分割すること）

```
game/
├── index.html          # エントリーポイント（HTMLのみ）
├── css/
│   └── style.css       # 全体のスタイル・アニメーション
├── js/
│   ├── main.js         # ゲーム初期化・モード選択
│   ├── scenes.js       # シーン管理（タイトル/モード選択/調理/結果）
│   ├── kitchen.js      # 自由キッチンモードのロジック
│   ├── shop.js         # バーガーショップモードのロジック
│   ├── cooking.js      # 調理アクション（焼く/盛り付け/トッピング）
│   ├── ui.js           # UI描画・食べ物の見た目・リアクション
│   ├── audio.js        # Web Audio API サウンド管理
│   └── data.js         # レシピ・食材・注文データ・セリフ
└── README.md           # 遊び方の説明
```

#### 分割の方針
- **1ファイル300行以下**を目安に分割する
- 各ファイルは単一の責務を持つ
- `data.js` にレシピや食材データを集約し、ロジックと分離する
- JSファイルはES Modules（import/export）で連携する
- `index.html` は `<script type="module" src="js/main.js">` で読み込む

## 🗺️ ゲーム構成

### タイトル画面

- 巨大な目だらけハンバーグ🍔👁️👁️👁️がドーンと表示
- 目がキョロキョロ動くアニメーション
- 「どうぞ召し上がれ！」のキャッチコピー
- 「はじめる」ボタン

### モード選択

プレイヤーは2つのモードから選ぶ：

#### 🏠 モード1：じゆうキッチン

好きな料理を自由に作れるモード。

**作れる料理：**
1. **目だらけハンバーグ**🍔👁️
   - ひき肉をこねる → 形を作る → 焼く → 目をつける → 盛り付け
2. **ポテト**🍟
   - じゃがいもを切る → 油で揚げる → 塩をふる
3. **お肉ステーキ**🥩
   - お肉を選ぶ → 焼き加減を決める → ソースをかける
4. **チーズ盛り合わせ**🧀
   - いろんなチーズを選んで盛り付ける

**調理の流れ（タップ操作）：**
1. 食材を選ぶ（タップで食材棚から取る）
2. 調理する（タップ連打で混ぜる、長押しで焼く、スワイプで切るなど）
3. トッピング（チーズ🧀・野菜🥬🍅🧅をドラッグで乗せる）
4. **目をつける**👁️（ハンバーグ専用：好きな数だけ目を乗せられる！）
5. 完成！→ 評価画面

**ハンバーグの目について：**
- 目👁️は好きなだけ乗せられる（1個〜100個！）
- 目の数が多いほどスコアが高い
- 目がたくさんあるとハンバーグが「世界一おいしい」認定される
- 目が動く（キョロキョロアニメーション）
- 見た目は怖いけど「せかいいち おいしい！」のテキスト表示で安心させる

#### 🍔 モード2：モスバーガー風 バーガーショップ

お客さんの注文通りにハンバーガーを作るモード。

**ゲームの流れ：**
1. お客さん😊が来店して注文する
   - 「ハンバーガーください！チーズ多めで！」
   - 「ポテトとハンバーガーのセットで！」
   - 「お肉ダブルでお願い！」
   - 「野菜たっぷりバーガー！」
2. 注文に合わせて材料を重ねていく
   - パン🍞（下）→ レタス🥬 → お肉🥩 → チーズ🧀 → トマト🍅 → パン🍞（上）
   - 材料をタップして順番に重ねる
3. 制限時間内に完成させる
4. 注文と合っていれば⭐評価！

**注文レベル：**
- Lv1: シンプルバーガー（パン＋肉＋パン）
- Lv2: チーズバーガー（パン＋肉＋チーズ＋パン）
- Lv3: デラックス（パン＋レタス＋肉＋チーズ＋トマト＋パン）
- Lv4: スペシャル注文（お客さんのこだわりオーダー）
- Lv5: 目だらけバーガー！👁️🍔👁️（最終チャレンジ）

**お客さんのリアクション：**
- 大成功⭐⭐⭐: 「せかいいち おいしい！！」🤩
- 成功⭐⭐: 「おいしい！ありがとう！」😊
- まあまあ⭐: 「うん、食べられるよ」😐
- 失敗: 「ちょっと違うかな…？」😅（でも怒られない。楽しく過ごすのが一番！）

## 🎨 ビジュアル・演出

- **スタイル**: ポップでカラフル。明るい背景色
- **キッチン背景**: カウンター、コンロ🔥、食材棚、お皿
- **ショップ背景**: カウンター、メニューボード、お客さん待ち列
- **食べ物の表現**:
  - 絵文字をベースに、目👁️をたくさん重ねて表示
  - 完成した料理は大きく画面中央に表示
  - 目だらけハンバーグ: 🍔の上に👁️が散りばめられたシュールな見た目
- **アニメーション**:
  - 焼く時: 🔥ジュージュー演出＋煙💨
  - 揚げる時: 油がパチパチ演出
  - 目をつける時: 目👁️がぴょんっと飛んでくる
  - 完成時: キラキラ✨＋「どうぞ召し上がれ！」テキスト
  - 目がキョロキョロ動く常時アニメーション

## 🔊 サウンド

- Web Audio APIで簡易サウンド:
  - ジュージュー音（焼く）
  - パチパチ音（揚げる）
  - トントン音（切る）
  - ポン！（食材を置く）
  - ぴょん！（目をつける）
  - ファンファーレ（完成）
  - 「いらっしゃいませ〜」チャイム（ショップモードでお客さん来店）

## 📊 ゲームシステム

- **自由キッチン**: スコアなし。自由に作って楽しむ。目の数でおもしろ評価
- **ショップモード**: ⭐評価制。注文との一致度＋スピードで⭐1〜3
- **コレクション**: 作った料理がアルバムに保存される
- **実績**: 「目100個ハンバーグ達成！」「10人のお客さんを満足させた！」など
- **難易度**: 失敗してもペナルティなし（楽しく過ごすのが一番！）

## 📱 必須UI要素

- タイトル画面（目だらけハンバーグのインパクトある表示＋「はじめる」ボタン）
- モード選択画面（「じゆうキッチン」と「バーガーショップ」）
- 調理画面（食材選択エリア＋調理エリア＋完成プレビュー）
- 完成＆評価画面（「どうぞ召し上がれ！」＋スコア）
- 料理アルバム画面（過去に作った料理一覧）
- ショップモード: 注文表示＋制限時間バー＋お客さんリアクション
